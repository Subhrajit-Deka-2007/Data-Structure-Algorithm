# Data Structure and Algorithm

A comprehensive learning repository for mastering **Data Structures and Algorithms** from scratch, implemented in **Java**.

## 📚 Overview

This repository is a dedicated learning resource designed to help you understand and master fundamental concepts of data structures and algorithms. Each section contains well-organized, clean implementations with explanations.

**Repository Created:** August 26, 2025  
**Primary Language:** Java  
**Status:** Active & Growing

## 📁 Contents

### Core Topics

- **[Bit Manipulation](BitManipulation/)** - Techniques for working with bits and binary operations
- **[Dynamic Programming](DynamicProgramming/)** - Solutions to optimization problems using memoization and tabulation
- **[Graphs](Graphs/)** - Graph data structures, traversals, and algorithms
- **[Greedy Algorithms](Greedy/)** - Greedy approach to solving optimization problems
- **[Prefix Sum Technique](Prefix_Sum_Technique/)** - Efficient range query and sum calculations
- **[Sliding Windows](Sliding_Windows/)** - Technique for solving subarray/substring problems

## 🎯 Learning Path

This repository covers essential algorithmic paradigms:

1. **Basic Techniques**
   - Bit Manipulation for efficient operations
   - Sliding Windows for linear problems
   - Prefix Sum for range queries

2. **Advanced Techniques**
   - Dynamic Programming for complex optimization
   - Graph algorithms and traversals
   - Greedy strategies for decision-making problems

## 🚀 Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- A code editor or IDE (VS Code, IntelliJ IDEA, Eclipse, etc.)

### How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Subhrajit-Deka-2007/DATA_STRUCTURE_AND_ALGORITHM.git
   cd DATA_STRUCTURE_AND_ALGORITHM
   ```

2. **Navigate to a topic:**
   ```bash
   cd BitManipulation  # or any other topic
   ```

3. **Compile and run Java files:**
   ```bash
   javac FileName.java
   java FileName
   ```
